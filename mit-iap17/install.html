dReal:
  • Follow instructions here.
  • Suppose you are using the dReal API in the file test.cpp. To compile:
    osx: g++ -std=c++11 test.cpp -I/usr/local/include/dreal -L/usr/local/lib/dreal -ldreal -libex -lnlopt -lcapd -lglpk -lprim -lClp -lCoinUtils -pthread
    ubuntu: g++ -std=c++11 -I/usr/include/dreal test.cpp -L/usr/lib/dreal -ldreal -libex -lnlopt -lcapd -lglpk -lprim -lClp -lCoinUtils -pthread
